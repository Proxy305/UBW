UBW: How did this come into being

话说买完iPad以后，我就发现我离不开它了。因为它成了我的移动资料库，以后读该死的PDF（不能像.doc文档那样可以用Web版式重排，手机看起来很不方便）就再也不用开台式机了，毕竟台式机是一开起来就功率近百瓦的怪物，而且我没办法走到哪里都带着台式机……不能带着它吃饭和上床是不是。

很快我又发现了一件让幸福感增大一个数量级的事情。学校图书馆原来很多书都有电子版！电！子！版！用图书馆的搜索系统找书，找出来的有些书旁边是有个小按钮的，一点下去就打开一个阅读器，就能读电子版了，就不用去图书馆借实体书了！说不定还可以将这本书下载到iPad，这样就更方便了！

可是下载的按钮到底在哪？我找了半天可是竟然一无所获。最后忘了是在图书馆的哪个网页上面看到说，图书馆的电子图书只提供在线阅读，不允许下载……然不允许下载……不允许下载…… （掀桌！！

然而前端攻城狮程序猿的经历告诉我，除非用控件，否则我看得到的东西，没有什么是我拿不到的。于是我花了一下午一晚上做出来了这个玩意，让我可以轻易地将文章用pdf格式下载下来，放到iPad里面慢慢读。

原来只是准备自己用，但是想了想还是把这个放出来吧，说不定对你们有帮助呢。（笑）

安装

提醒：现阶段这个扩展程序只提供适用于谷歌浏览器-Google Chrome的版本，如果你没有安装谷歌浏览器，你必须先安装以使用UBW。适用于各种浏览器的版本正在开发中，敬请期待。

  1. 首先请先到这里下载UBW扩展程序。
  2. 启动Google Chrome 浏览器。找到地址栏右侧的 按钮，点开浏览器菜单，选择“设置”，或者直接在浏览器地址栏内输入：chrome://settings
  3. 找到设置界面左上的“扩展程序”，点击，进入扩展程序界面，将下载好的扩展程序拖进这个界面。在出现的对话框里面选“是”。
  4. 这时UBW扩展程序已经安装完成，可以使用了。

使用说明

  1. 安装完成后，你会在地址栏右侧找到UBW的图标。
  2. 进入暨南大学图书馆图书搜索页http://opac.jnu.edu.cn/*chx，搜索你想看的书。
  3. 点开结果列表中的一本书，如果书名右侧有“阅读电子版”的小按钮，说明这本书有电子版。点击该按钮，打开阅读器界面。
  4. 进入阅读器后，用鼠标滚轮一页一页地向下翻页，直到翻到最后一页。注意不要翻得太快，要确保每一页都显示出来了再翻页。但也不要翻得太慢，保持一个速度滚滚轮，恰好能让每一页在移出视区之前恰好显示出来就行了。
  5. 高声咏唱页面顶端的咒语，发动Unlimited Book Works。（大雾）
  6. 点击UBW图标打开UBW扩展程序，点击“Generate”按钮，UBW就会生成一份文档预览。如果文档预览正常，就可以点击UBW界面上的“Print”按钮生成PDF文件了。
  7. 在弹出的窗口当中“边距”选择“无”，目标打印机选择“保存为PDF”。设置完成后，点击“保存”，在弹出的对话框中选择文件的保存地址就可以了。

FAQ

  1. 我发现生成的文件里面页面太小或者太大，怎么办？

    请使用UBW的“缩放”选项。在点击“Generate”生成预览之前，请先勾选主界面上的“Scaling”选项，并在旁边的输入框里面输入一个合适的缩放指数。预设的缩放指数是90%。

  2. 生成的PDF我没有办法拷贝里面的文字！

    很遗憾，这件事情神仙也办不到。图书馆提供的电子书本来就是扫描版，扫描版是没有办法选择里面的文字的。用Adobe Acrobat之类的软件识别里面的文字试试吧。

  3. UBW能够下载百度文库的文章吗？

    不能。UBW只能下载暨南大学图书馆电子书在线阅读的内容。无论是对百度文库也好，对其他学校的电子书

  4. 生成的文章里面有空白页是怎么回事？

    那是因为你在步骤3翻页的时候翻的太快了。务必一页一页的翻，不要用跳转。

  5. 为什么需要一页一页地翻页啊？

    这和学校的电子书阅读器是有关的，只有你一页一页的翻页，UBW才能正确抓取每一个页面。这个过程可以用鼠标滚轮以一个适合的速度向下滚，一般一分多种可以滚完一本书，还是不算太慢的。毕竟发动技能需要咏唱时间嘛是不是？;-)

  6. 我发现了一个BUG！我应该告诉谁？

    你可以直接联系我，虽然我并不推荐这样做。如果你有Github账号的话，我推荐你前往UBW的项目主页（见下文）Report一个Issue。
